Component Usage Guide

Key patterns and conventions used throughout the app.

1. Tier Color System

Always use the tier modifier class. Never hardcode tier colors — they change as the user progresses.

/* CORRECT — use modifier class */ <div class="user-card user-card--silver">...</div> <div class="status-card status-card--gold">...</div> <span class="pill pill--tier pill--tier-bronze">Бронза</span> /* Available modifiers: --bronze | --silver | --gold */ /* For progress dots on light bg, match the user's tier: */ <span class="prog-dot prog-dot--silver"></span> /* user is silver */ /* For progress dots ON the gradient card, always use white: */ <span class="prog-dot prog-dot--card-filled"></span> <span class="prog-dot prog-dot--card-empty"></span>

2. Glass Metric Pattern

Metric boxes on gradient cards use glassmorphism. This is achieved via background: rgba(255,255,255,0.18) + backdrop-filter: blur(4px). Both .user-card__metric and .status-card__metric use this pattern.

/* Glass metric inside gradient card */ .user-card__metric { background: rgba(255,255,255,0.18); backdrop-filter: blur(4px); border-radius: var(--r-md); padding: var(--sp-sm) var(--sp-md); } /* Key difference between user-card and status-card metrics: user-card → value on top, label below status-card → label on top, value below */

3. Navigation Active State

Active nav item is indicated by a 2px blue line at the top (via ::before pseudo-element) and brand-blue color. Toggle by adding/removing .active class.

// JavaScript: switch active tab document.querySelectorAll('.bottom-nav__item').forEach(item => { item.classList.remove('active'); }); document.querySelector('[data-tab="status"]').classList.add('active');

5. Responsive / Layout Notes

The app is a mobile-first single-column layout, max-width 430px, centered. Key layout variables:

--max-width: 430px; /* App shell max width */ --nav-height: 60px; /* Bottom nav height (excludes safe area) */ /* Most sections use: */ margin: 0 var(--sp-base) var(--sp-lg); /* = margin: 0 16px 20px */ /* Screen padding-bottom to clear the nav: */ padding-bottom: calc(var(--nav-height) + env(safe-area-inset-bottom, 0px) + 16px);
